Input Sequence Layer

Representa el historial cronológico de compras o interacciones de un usuario con los juegos. Los IDs de juegos se introducen como enteros secuenciales indexados. Se reserva el ID 0 estrictamente para representar el padding (relleno) cuando la secuencia es más corta que la longitud máxima requerida.
